完整功能矩阵

多 AI 并行查询

核心功能

同时调用多个 AI 模型，各模型独立 SSE 流式输出。支持 Kimi、Claude、Codex、DeepSeek、GPT-4o。

🔍

自动账号检测

Auth Detector

启动时扫描 7 类来源，自动发现本机已登录的 AI 账号，无需手动配置 API Key。

📥

素材库管理

Material Library

选中任意文字一键收藏，右侧素材库面板统一管理，支持选择多个素材进行评审总结。

📊

评审总结

Review Engine

手动触发评审总结，系统调用评审模型对所有素材和 AI 响应进行智能对比分析。

🛡️

CLI 安全审批

CLI Sandbox

敏感 CLI 命令执行前弹窗确认，5 级风险评估，子进程沙箱执行，审计日志记录。

⚙️

管理控制台

Management

Skills 技能配置、CLI 运行时管理、模型参数调整、MCP 服务器配置，SQLite 持久化保存。

📜

历史任务

Task History

三栏式任务管理：任务列表 / 时间线 / 素材。所有查询历史自动保存，可随时回顾。

💾

SQLite 持久化

Data Layer

所有数据本地 SQLite 存储，自动保存机制，数据不丢失，隐私安全。

🖥️

Electron 桌面

Desktop App

Electron 42 打包为 macOS 应用（.dmg）。后端作为子进程启动，自动等待健康检查。
